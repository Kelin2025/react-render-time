# react-render-time

React hook to check how much does it take to render

## Installation

```
yarn add react-render-time
```

## Usage

```js
import React from "react"
import useRenderTime from "react-render-time"

const App = () => {
  userRenderTime(name)

  return (
    <div>
      <h1>Hello there</h1>
    </div>
  )
}
```
